# 📝 Word Counter

A simple and responsive **Word Counter** web application built using **HTML, CSS, and JavaScript**. It provides real-time text analysis by counting the number of words, letters, and spaces as users type or paste text.

## ✨ Features

- ✅ Real-time word counting
- ✅ Letter (character) counting
- ✅ Space counting
- ✅ Clean and responsive user interface
- ✅ Fast and lightweight
- ✅ Works in all modern web browsers

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 🚀 How It Works

1. Open the application in your browser.
2. Type or paste text into the text area.
3. The application instantly displays:
   - Total Word Count
   - Total Letter Count
   - Number of Spaces
